1수집하는 정보

HabitBloom은 아래 정보만을 사용자의 기기 내부 저장소에 보관합니다. 외부 서버로 전송되거나 클라우드에 동기화되지 않습니다.

2정보 이용 목적

  • 습관 목록 표시 및 완료 처리
  • 스트릭(연속 달성일) 및 통계 계산
  • 다크모드 등 사용자 환경 설정 유지

수집된 정보는 위 목적 외 다른 용도로 사용되지 않습니다.

3보관 위치 및 보유 기간

모든 데이터는 사용자 기기 내부 저장소(SharedPreferences)에만 보관됩니다.

  • 외부 서버 전송: 없음
  • 클라우드 동기화: 없음
  • 제3자 전달: 없음
  • 보유 기간: 사용자가 직접 삭제하거나 앱을 제거할 때까지

4인터넷 및 권한

HabitBloom은 완전 오프라인 앱이며 어떠한 시스템 권한도 요청하지 않습니다.

INTERNET 권한 없음 위치 권한 없음 카메라/마이크 없음 연락처/캘린더 없음 알림 권한 없음

5광고 및 분석 SDK

광고 SDK 없음 Firebase Analytics 없음 Adjust / AppsFlyer 없음 크래시 리포팅 SDK 없음

제3자 분석/광고 트래커가 일체 포함되지 않습니다.

6사용자 권리

사용자는 언제든지 아래 방법으로 본인의 데이터를 완전히 삭제할 수 있습니다.

  • 앱 내 설정 → 모든 데이터 초기화 메뉴 사용
  • 기기에서 앱을 삭제 (앱과 함께 모든 로컬 데이터가 제거됩니다)

7알림 기능

현재 버전(v1.0.1)에는 로컬 알림 기능이 포함되어 있지 않으며, 알림 권한도 요청하지 않습니다.

8아동 개인정보 보호

HabitBloom은 14세 미만 아동의 개인정보를 별도로 수집하지 않습니다. 본 앱은 어떠한 개인정보도 수집하지 않으므로 연령에 관계없이 안전하게 이용 가능합니다.

9정책 변경

개인정보 처리방침이 변경될 경우 앱 내 공지 또는 본 공개 URL을 통해 사전 고지합니다. 중요한 변경 시 앱 업데이트 시점에 사용자에게 안내됩니다.
